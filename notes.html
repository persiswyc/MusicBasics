【Notes】

Whole Note

It looks like a ring.
A whole note equals to 4 beats.

Half Note

It looks like a ring with a stick.
A half note equals to 2 beats.

Quarter Note

It looks similar to a half note but the ring is filled.
A quarter note equals to 1 beat.

Eighth Note

A eighth note equals to 0.5 (half) beat.

Sixteenth Note

A sixteenth note equals to 0.25 (1/4) beat.
